Artificial Neural Networks

This is where I really get excited. Neural networks are proving to be a popular trend at the moment and there's no question why. Programmed to simulate the neural network of a biological brain, in a way an artificial neural network is a framework of self-learning algorithms. The main difference between an algorithm and a neural network is that a neural network can predict something it hasn't sceen before. Training neural networks requires a huge amount of processing power which isn't always easy to comeby, luckily we have Google who has developed Tensorflow JS, a library that allows neural networks to be trained in the browser via the GPU. To the left you can see a simple demo of a neural network training off of classified RGB values in a database. Change the color on the screen and watch the browser attempt to predict your chosen color.
